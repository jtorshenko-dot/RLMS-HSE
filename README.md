# RLMS-HSE Data Analysis

Проект, посвящённый анализу данных **RLMS-HSE** с помощью Python. В работе выполняются загрузка, очистка, гармонизация и объединение социологических данных, а затем исследовательский анализ социально-демографических показателей.

## About the project

Проект показывает полный цикл работы с реальными табличными данными: от предварительной обработки и рекодирования переменных до визуализации и интерпретации результатов. В ноутбуке объединяются индивидуальные и домохозяйственные анкеты, после чего анализируются характеристики респондентов и домохозяйств.

## Tools
Python: pandas, numpy, matplotlib, seaborn, pyreadstat
